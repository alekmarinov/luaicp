transform (A, B)
Computes rotation R and translation t aligning A to B such that min|(R*A-t) - B|

Parameters

  • A: is the source set to align from
  • B: is the target set to align to

Return values:

  1. rotation R
  2. translation t
  3. details table with internaly computed results, e.g. the centroids of A and B
