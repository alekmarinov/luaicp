Functions

one2many (A, B, distance, wA, wB)
Computes subset of B - the closest points to A such as each element of A can have one or more corresponding points in B

Parameters

  • A: is the set to be mapped to
  • B: is the set to be mapped from
  • distance: is a distance function
  • wA: optional scalar weights of the points in A
  • wB: optional scalar weights of the points in B

Return values:

  1. subset of B
  2. subset of wB or nil
