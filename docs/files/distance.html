Functions

euclid (a, b)
Computes the euclid distance between vector a and vector b

Parameters

  • a: is the 1st vector
  • b: is the 2nd vector such as |a| = |b|

Return value:

||a-b||
square (a, b)
Computes the square distance between vector a and vector b

Parameters

  • a: is the 1st vector
  • b: is the 2nd vector such as |a| = |b|

Return value:

Sum (a[i]-b[i])^2, i = 1..|a|
weighted (a, b, wa, wb)
Computes a weighted distance between vector a and vector b
if input weights are omited the result distance is the same as euclid

Parameters

  • a: is the 1st vector
  • b: is the 2nd vector such as |a| = |b|
  • wa: is optional scalar weight for vector a
  • wb: is optional scalar weight for vector b

Return value:

(Max(wa, wb)/Min(wa, wb))^2 * ||a-b||
