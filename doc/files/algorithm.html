iterator (A, B, map, distance, wA, wB)
Approaches set A to set B by iterative translations and rotations

Parameters

  • A: is the set to be moved
  • B: is the target set for A movements
  • map: is a user defined function which returns the subset of B - the closest points to A.
  • distance: is a user defined function which returns the distance estimation for 2 arbitrary points
  • wA: is optional point weights if set A
  • wB: is optional point weights if set B

Usage:

for A, Bs, R, t, wBs in algorithm(A, B, icp.map.one2many, icp.distance.euclid) do ... end

Return value:

iterator function
